Bulk Add Tokens

Paste a list of tokens. Each line can be single-sided or double-faced; the parser decides by how many fields you supply.

Format

One line per physical token. Whitespace-separated. Field count picks the line type:

  • set collector — single-sided (uses default qty)
  • set collector qty — single-sided with explicit qty
  • front_set front_# back_set back_# — DFC (uses default qty)
  • front_set front_# back_set back_# qty — DFC with explicit qty

Blank lines and lines starting with # are ignored. 

# Single-sided
tsoc 27
# Single-sided with quantity
thob 12 6
# DFC, front + back from different sets
tblb 3 tblc 14
# DFC, both sides same set
tblb 3 tblb 7
# DFC with explicit quantity
tblb 1 tblc 8 2
Cancel

Each pair triggers two Scryfall lookups; ~80ms throttled. Expect roughly ~0.2 seconds per token. A 100-row paste takes ~20 seconds.
