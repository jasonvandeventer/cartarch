Bulk Add DFC Tokens

Paste a list of double-faced token pairs. Each line is the front and back, looked up from Scryfall.

Format

One line per physical token. Whitespace-separated: front_set front_collector back_set back_collector [quantity]. Quantity defaults to the value below if omitted. Blank lines and lines starting with # are ignored. 

# Front and back from different sets
tblb 3 tblc 14
# Same front paired with a different back
tblb 3 tblb 7
# With explicit quantity
tblb 1 tblc 8 2
Cancel

Each pair triggers two Scryfall lookups; ~80ms throttled. Expect roughly ~0.2 seconds per token. A 100-row paste takes ~20 seconds.
